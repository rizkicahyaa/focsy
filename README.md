# Focsy - Pomodoro Timer App

Focsy adalah aplikasi **Pomodoro Timer berbasis web** yang membantu kamu mengatur waktu fokus dan istirahat secara seimbang. Didesain dengan tampilan modern, efek blur lembut, dan sistem penyimpanan otomatis agar progres tidak hilang meskipun halaman direload.

## Fitur Utama

-   **Timer Fokus & Istirahat Otomatis**  
    Setelah waktu fokus selesai, aplikasi otomatis berpindah ke waktu istirahat, dan sebaliknya.

-   **Notifikasi & Suara Peringatan**  
    Saat sesi berakhir, kamu akan mendapat notifikasi desktop dan suara alert.

-   **Auto Save dengan LocalStorage**  
    Semua data — termasuk waktu tersisa, mode (focus/break), status timer, dan catatan — akan disimpan otomatis.

-   **Catatan Sederhana (Noted)**  
    Fitur kecil untuk mencatat hal penting selama sesi Pomodoro.

-   **Desain Responsif & Estetik**  
    Tampilan elegan dengan efek blur, warna lembut, dan tetap nyaman di berbagai ukuran layar (HP, tablet, laptop).

## Teknologi yang Digunakan

-   **Vite**
-   **React + TypeScript**
-   **Tailwind CSS**

## Cara Menjalankan

### 1. Clone repository

```bash
git clone https://github.com/rizkicahyaa/focsy.git
cd focsy
```

### 2. Menjalankan npm

```bash
npm install
```

### 3. Menjalankan aplikasi

```bash
npm run dev
```
